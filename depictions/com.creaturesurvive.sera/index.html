Supports iOS < 9.0

Sera is a complete LockScreen customization tweak that allows you to redesign your entire LockScreen. Sera is currently in beta and the feature set is not complete, you may encounter bugs. If you find any bugs please contact me via support in settings or via Cydia.

Features

  • Date and time positioning
  • Date and time height
  • Date and time size
  • Date and time background color
  • Date and time background alpha
  • Date and time font
  • Date and time font size

  • Media controls position
  • Media controls size
  • Media controls compact view

  • Notification area position
  • Notification area height
  • Notification area background color
  • Notification area background alpha
  • Notification area background blur
